## 📖 專案簡介 (About)

**HCFD 事故安全官作業平台 (Incident Safety Officer Platform)** 是一個專為消防與救災現場「事故安全官 (ISO)」所開發的私人輔助決策系統。
本平台整合了多項災害現場不可或缺的風險評估工具與資訊，旨在協助安全官於分秒必爭的救災環境中，快速掌握現場危害因素、氣候變遷及風險模型，進而保障全體救災人員的生命安全。

## ✨ 核心功能模組 (Core Features)

本平台目前包含以下四大核心作業模組：

### 1. ☣️ Hazmat (危害物質處置支援)
提供危險化學品、毒災等危害物質 (Hazardous Materials) 的相關處置指引與安全距離建議，協助安全官快速查詢物質特性並劃定警戒區。

### 2. 📋 ISO Guide (安全官作業指南)
內建事故安全官的標準作業程序 (SOP)、檢核表與安全準則，確保現場安全官能依循標準化流程執行任務，避免人為疏漏。

### 3. 🌦️ 環境氣候監控及 NOAA (氣象與環境監控)
介接環境與氣象數據（含 NOAA 相關資源），提供現場風向、風速、溫濕度及劇烈天氣預警，為戶外救災及有毒氣體擴散預測提供關鍵的環境參數。

### 4. 📊 風險決策模型 (Risk Decision Model)
透過量化的風險評估模型，協助指揮官與安全官進行現場風險/效益分析。在「救人」與「防護」之間取得最佳平衡，產出具體的戰術安全建議。

## 🚀 平台使用與執行 (Usage)

本系統無需安裝任何後端伺服器，完全基於網頁瀏覽器運行，適合在平板、筆電或手機端快速開啟：

1. 透過瀏覽器開啟 [平台網址](https://fc861117-sketch.github.io/HCFD_ISO_PLATFORM/)。
2. 於首頁導覽列 (Navigation) 點選對應的決策模組即可開始使用。

*(註：若要進行本地端測試，只需 clone 此專案並在瀏覽器中開啟 `index.html` 即可。)*
